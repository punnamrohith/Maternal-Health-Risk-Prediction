
# Maternal Health Risk Prediction

## Project Description
Maternal Health Risk Prediction is a machine learning project that analyzes health data of pregnant women to predict the level of risk during pregnancy. The system uses features like age, blood pressure, blood sugar level, body temperature, and heart rate to determine whether the risk is Low, Medium, or High.

## Objective
The objective of this project is to help identify potential health risks in pregnant women at an early stage so that proper medical care can be provided on time.

## Dataset Features
- Age
- Blood Pressure
- Blood Sugar Level
- Body Temperature
- Heart Rate

## Technologies Used
- Python
- Machine Learning
- Data Analysis

## Result
The model predicts the maternal health risk level as Low, Medium, or High. If a high risk is detected, the system can notify the user so that early medical attention can be taken.

## Conclusion
This project helps in early detection of maternal health risks using machine learning techniques, which can improve healthcare monitoring during pregnancy.
